# Go RTC App

The purpose of this project is to learn more about the workings of GO as a backend system.

## Learning Objectives
1. GO
2. Backend programming
3. Basic Front end programming


## To Run Project
- Clone project
- cd to backend folder
- Build dockerized backend with this command
```
docker build -t backend .
```
- Run backend with command
```
docker run -it -p 8080:8080 backend
```
- cd to frontend folder
- Start frontend with this command
```
npm start
```
